FREQUENTLY ASKED QUESTIONS


Q: WHEN WILL THE PROJECT BE COMPLETED?


A: We currently don't have any release date for public testing, as the project is pretty much bare-bones. However, we are really inspired and doing our best, so be patient! As for when will the project end, it's a tricky question on its own. League of Legends is constantly udpated just as Minecraft is, so support for future version is an objective we have in mind. In the case of Minecraft updates introducing conflicts or modifiying critical mechanics of the project, we don't know if we would be able to continue. We'll see.

Q: WHICH SOFTWARE ARE YOU USING FOR THE DEVELOPMENT OF THIS PROJECT?


A: We use Sublime Text 3 and Atom for coding, Blockbench for modelling and both Aseprite and Paint.NET for image manipulation.

And of course, Minecraft!

Q: DO YOU HAVE ANY FUNDING?


A: No! We aren't receiving a dime for this project but any personal donations would be highly appreciated! Donate via paypal here, thank you ❤.

The final product won't cost anything, either.

Q: WHERE CAN I KEEP MYSELF UPDATED WITH THE PROJECT? HOW CAN I CONTACT YOU?


A: If you're too busy to constantly check our main page, Noelia is in charge of our twitter account, so she should be posting updates regularly once the project picks up good pace! Don't be afraid to DM us!.

Additionally, you can always check our contact page if any more private issue arises.

Q: DO YOU OWN THE RIGHTS TO LEAGUE OF LEGENDS INTELLECTUAL PROPERTY?


A: Of course not! LOLIMProject was created under Riot Games' "Legal Jibber Jabber" policy using assets owned by Riot Games. Riot Games does not endorse or sponsor this project.

Aditionally, in case Riot Games contacts us, we will immediately cease any work.
